________MOTIVATION________


When we are using PowerPoint, we often come in a situation about what the proverb that we seen means.

When we are using a forenign language it sometimes confused us when and how to use a proverb properly to make us sound like a native speaker.

When we are using translate applications, we found that for some reasons, the system will mistranslate some specifc words or pharses we used in the sentence, we wondered what cause this situation and how does the translate application translates.
